## Getting Started

```sh
# package install
$ npm i

# Start a server
$ npm run dev # you can access http://localhost:7777

# auto format
$ npm run fmt

# package build
$ npm run build

# test
## spec
$ npm run test:spec # pr npm run test:spec:watch

## feture
$ npm run test:feature

# storybook
$ storybook
```
